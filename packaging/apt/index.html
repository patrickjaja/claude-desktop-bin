Claude Desktop for Linux

Unofficial APT repository for claude-desktop-bin.

Quick Setup (Debian / Ubuntu)

curl -fsSL https://patrickjaja.github.io/claude-desktop-bin/install.sh | sudo bash
sudo apt install claude-desktop-bin

Manual Setup

# Import GPG key
curl -fsSL https://patrickjaja.github.io/claude-desktop-bin/gpg-key.asc \
  | sudo gpg --dearmor -o /etc/apt/keyrings/claude-desktop.gpg

# Add repository
echo "deb [signed-by=/etc/apt/keyrings/claude-desktop.gpg arch=amd64] https://patrickjaja.github.io/claude-desktop-bin/deb/ ./" \
  | sudo tee /etc/apt/sources.list.d/claude-desktop.list

# Install
sudo apt update && sudo apt install claude-desktop-bin

Updates

Once installed, updates arrive through your normal system updates:

sudo apt update && sudo apt upgrade

Other Distros

  • Arch Linux / Manjaro: yay -S claude-desktop-bin (AUR)
  • AppImage: Download from GitHub Releases
This is an unofficial community package, not supported by Anthropic.
