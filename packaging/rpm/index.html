Claude Desktop for Linux

Unofficial RPM repository for claude-desktop-bin.

Quick Setup (Fedora / RHEL / openSUSE)

curl -fsSL https://patrickjaja.github.io/claude-desktop-bin/install-rpm.sh | sudo bash
sudo dnf install claude-desktop-bin

Manual Setup

# Import GPG key
sudo rpm --import https://patrickjaja.github.io/claude-desktop-bin/gpg-key.asc

# Add repository
sudo tee /etc/yum.repos.d/claude-desktop.repo <<'EOF'
[claude-desktop]
name=Claude Desktop for Linux
baseurl=https://patrickjaja.github.io/claude-desktop-bin/rpm/
enabled=1
gpgcheck=1
gpgkey=https://patrickjaja.github.io/claude-desktop-bin/gpg-key.asc
repo_gpgcheck=1
EOF

# Install
sudo dnf install claude-desktop-bin

Updates

Once installed, updates arrive through your normal system updates:

sudo dnf upgrade claude-desktop-bin

Other Distros

  • Debian / Ubuntu: APT repository
  • Arch Linux / Manjaro: yay -S claude-desktop-bin (AUR)
  • AppImage: Download from GitHub Releases
This is an unofficial community package, not supported by Anthropic.
